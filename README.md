# MediVolt - Medical Supply Demand Forecasting

MediVolt is a medical supply demand forecasting application that helps healthcare organizations predict and manage their supply requirements using machine learning.

## Features

- **User Authentication**: Login, signup, and password management
- **Dashboard**: Visual analytics and insights
- **ML Model**: Demand forecasting using machine learning
- **Dataset Management**: View and manage medical supply data
- **Evaluation**: Model performance analysis and visualization

## Tech Stack

- **Backend**: Python (Flask)
- **Database**: SQLite
- **Frontend**: HTML, CSS, JavaScript
- **ML**: Scikit-learn, Pandas, NumPy

## Installation

1. Install Python dependencies:
```
bash
pip install -r requirements.txt
```

2. Run the application:
```
bash
python app.py
```

3. Open `index.html` in your browser or access at `http://localhost:5000`

## Project Structure

- `app.py` - Main Flask application
- `database.py` - Database utilities
- `ml_model.py` - Machine learning model
- `ml_models.py` - Additional ML models
- `evaluate_model.py` - Model evaluation
- `view_dataset.py` - Dataset viewing utilities
- `view_db.py` - Database viewing utilities
- `requirements.txt` - Python dependencies
